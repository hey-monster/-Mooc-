线性表：同类元素的有序序列
数组
```
typedef struct LNode *List;
struct LNode{
    ElementType Data[MAXSIZE];
    int Last;
};
```
链表
```
typedef struct LNode *List;
struct LNode{
    ElementType Data;
    List Next;
};
struct Lnode L;
List PtrL;
```
广义表
```
typedef struct GNode *GList;
struct GNode{
    int Tag; /*标志域：0单元素，1广义表*/
    union {
        ElementType Data;
        GList SubList;
    }URegion;
    GList Next;
};
```
